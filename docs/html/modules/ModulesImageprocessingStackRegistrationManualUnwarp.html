Unwarp images (manual)

Description

Uses the bUnwarpJ_ plugin by Ignacio Arganda-Carreras. Arganda-Carreras, I., et al., CVAMIA: Computer Vision Approaches to Medical Image Analysis (2006), 85-95

Parameters

  • Input image (default = "")

  • Output image (default = "")

  • Reference image (default = "")

  • Point selection mode (default = "Select at runtime")

  • Registration mode (default = "Fast")

  • Subsample factor (default = "0")

  • Initial deformation mode (default = "Very Coarse")

  • Final deformation mode (default = "Fine")

  • Divergence weight (default = "0.0")

  • Curl weight (default = "0.0")

  • Landmark weight (default = "0.0")

  • Image weight (default = "1.0")

  • Consistency weight (default = "10.0")

  • Stop threshold (default = "0.01")

  • Fill mode (default = "Black")

  • Enable multithreading (default = "true")
